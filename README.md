# gvm

golang version manage

`one issue need help: can not use in goland's terminal!`

## init

```shell
gvm init
```

## install go

```shell
# will install go 1.18
gvm install go1.18

# will install go 1.18.5 version
gvm install go1.18.5
```

## activate go

```shell
# will activate go 1.18 environment
gvm activate go1.18
```
